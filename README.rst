Wake up mechanism


Installation
============

Install by running::

    pip install Mopidy-WakeUp

Or, if available, install the Debian/Ubuntu package from `apt.mopidy.com
<http://apt.mopidy.com/>`_.


Configuration
=============

Mopidy-WakeUp to your Mopidy configuration file::
